\graffito{El desarrollo de este proyecto requirió conocimientos de Desarrollo de Software, Bases de Datos Relacionales, Aplicaciones Web, Microservicios, Seguridad y Servicios en la Web, Redes de Computadores y Sistemas Operativos.\bigskip}

\section{Justificación Económica}
Actualmente no existe un proyecto \emph{Open Souce} que permita aprovisionar recursos en \acrshortpl{CCSP} e integrar la validación de sus costos; gracias a que \appName es un proyecto de código abierto se puede implementar de manera gratuita por cualquier desarrollador para dar un punto de apoyo en este proceso que carece de un marco bien definido.\bigskip

\section{Justificación Académica}
Diseñar e implementar esta propuesta puso a prueba las destrezas técnicas y formación del aspirante a grado; Este proyecto de código libre permite tener un punto de partida para que mas interesados puedan contribuir en la solución del problema que parece tener carencia de participación.
